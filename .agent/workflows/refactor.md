# Workflow: Patterns Refactor
## Mission (指令内容)
将当前代码从“过程式编程”重构为“模式驱动编程”。
1. **配置化**：将硬编码的阈值（如 800ml）抽离到系统配置中。
2. **解耦**：将业务逻辑从 UI 组件中抽离到 Custom Hooks 中。

## AI Behavior (执行标准)
- **对比展示**：使用 Diff 视图清晰展示重构前后的差异。
- **重构原则**：遵循 DRY (Don't Repeat Yourself) 和 SOLID 原则。

## Learning Points (学习引导)
- 解释本次重构使用了哪种重构手法（例如：Extract Function 或 Replace Magic Number with Symbolic Constant）。
