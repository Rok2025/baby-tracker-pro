# Workflow: Mentor Review
## Mission (指令内容)
请对当前选中的代码或文件进行深度审计。
1. **逻辑漏洞**：寻找边界条件处理（如跨天时间计算）和数据一致性风险。
2. **架构合规**：检查是否绕过了 @lib/supabase.ts 的封装。
3. **性能建议**：分析是否存在不必要的 React 重新渲染。

## AI Behavior (执行标准)
- **不要直接修复**：先列出问题清单。
- **苏格拉底式提问**：针对严重问题，先问我“你觉得这里如果遇到[边界情况]会发生什么？”，引导我思考。

## Learning Points (学习引导)
